# Odoo Starter Pack

Repository ini adalah untuk melakukan development odoo secara local menggunakan docker

### Step-step

1. Pastikan pada perangkat sudah terinstall docker desktop.
2. Sesuaikan konfigurasi di `./etc/odoo/odoo.conf` ganti password sesuai dengan kebutuhan.
3. Setelah itu buka terminal dan jalankan `docker compose up`
4. Tunggu hingga selesai dan buka http://localhost:10018 jika berhasil maka akan ada tampilan awal untuk setup odoo.
